# Cravio

Full-stack food ordering and restaurant management web application built using Django, PostgreSQL, HTML, CSS, and JavaScript.

## Live Demo

https://cravio-zz6e.onrender.com/

---

## Features

- User Authentication
- Restaurant & Menu Management
- Food Ordering System
- Shopping Cart
- Checkout Flow
- Responsive UI
- Admin Dashboard
- PostgreSQL Database Integration
- Cloud Deployment

---

## Tech Stack

### Backend
- Python
- Django

### Frontend
- HTML
- CSS
- JavaScript

### Database
- PostgreSQL

### Deployment
- Render

---

## Run Locally

```bash
git clone https://github.com/CRSanju/Cravio.git

cd Cravio

python manage.py runserver
```

---

